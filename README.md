# XL Launcher

A lightweight, open-source Minecraft launcher for users seeking more flexibility, performance, and customization. Multiple mod loaders, extended system-level adjustments for increased stability, and offline and online launching are all supported by XL Launcher.

Important: Work on XL Launcher is still ongoing. Certain indexing features, memory optimizations, and background processes are experimental.
